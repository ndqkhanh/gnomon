The three gaps Gnomon closes

G1 · Harness IR

No LLVM-equivalent exists for agent harnesses in 2026. 14+ frameworks describe the same loop-over-tools-with-memory-and-guards in incompatible formats.

G2 · Primitive attribution

Trace stores say "the trace failed." None say "compaction dropped the fact on turn 7" or "permission blocked a legitimate action." You can't improve what you can't attribute.

G3 · Chaos for agents

Classical chaos engineering exists for infrastructure. For agents, no framework injects tool-latency spikes, stale memory reads, or compaction-induced fact loss to stress-test recovery paths.

How a failure becomes a patch

  1. Claude Code session emits a trace export. Adapter normalises into HIR events.
  2. Trace store appends hash-chained; validates DAG integrity.
  3. HAFC attributes each failure to a primitive (e.g. skill_miss on weekly_review).
  4. Evolution loop proposes an Autogenesis-shaped ResourcePatch (add a trigger phrase to the skill).
  5. Replay sandbox re-executes the trace with the patched resource; HAFC compares attribution deltas.
  6. Gate decides: Pass^k non-regression, attribution volume dropped, mesa-guard clear, second-family judge agrees.
  7. Commit → resource DAG advances; bundler emits Claude Code SKILL.md (and, on roadmap, Cursor / LangGraph / LobeHub / OpenClaw artefacts).
  8. 72h rollback window watches for regression.

Design grounding

  • Attribution-as-SEA-reward. Voyager uses env success, Hermes uses skill-reuse, Autogenesis uses a gated evaluator — every SEA system reinvents reward. Gnomon's primitive-attribution is the first shared reward signal. Anchors: docs 19, 36, 55, 56, 57, 58, 59.
  • Cross-channel evidence. Inherited from Vertex-Eval. An attribution is confirmed only when ≥ 2 of {trace, audit, state-snapshot} agree. Anchor: doc 38.
  • HORIZON-style attribution. Primitive + quoted step + reproducer. Anchor: doc 27.
  • Chaos for agents. Population-reliability depends on decorrelated failures. Anchors: docs 49, 53.
  • Meta-harness positioning. Gnomon does not compete with authoring (Archon, DeerFlow, LangGraph) — it consumes their traces and ships improvements back. Anchor: doc 66.
